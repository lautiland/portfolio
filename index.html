Sobre mí

Tengo 22 años y estudio en la Facultad de Ingeniería de la Universidad de Buenos Aires. Mi interés por la tecnología surgió durante la adolescencia al tomar cursos de programación en Salta, lo que definió mi camino profesional. En la UBA he explorado áreas como algoritmos, arquitectura de software y desarrollo de sistemas, consolidando mi enfoque en soluciones eficientes y escalables.

Este portfolio incluye trabajos prácticos universitarios donde apliqué estructuras de datos, patrones de diseño y bases de datos. Muchos son proyectos colaborativos desarrollados con Git y documentación detallada, demostrando cómo traslado conceptos teóricos a soluciones prácticas. Busco oportunidades para seguir creciendo profesionalmente mientras completo mi formación.

Proyectos

Algo Roma

Juego de tablero por turnos con temática de gladiadores y casilleros con modificadores.

Trabajo práctico universitario desarrollado colaborativamente en Java con Maven, aplicando el patrón MVC. Implementamos Test Driven Development con énfasis en la interacción entre objetos, diseñando un sistema de modificadores dinámico para algunos casilleros. La arquitectura incluye diagramas UML de clase y secuencia que documentan las mecánicas centrales.

Algo Hoot

Juego de preguntas por turnos con distintas tipos de pruebas, categorías y modificadores.

Trabajo práctico universitario colaborativo desarrollado con Java Maven bajo arquitectura MVC. Implementamos Test Driven Development para crear el proyecto, aplicando patrones de diseño que gestionan la diversidad de pruebas y modificadores. La documentación con diagramas UML de clase y secuencia modela respalda la arquitectura.

Scouteando

Página web estilo blog acerca del movimiento scout que muestra variedad de artículos con sus respectivas categorías.

Proyecto personal que desarrollo de forma autodidacta, iniciado con HTML, CSS y JavaScript, y posteriormente migrado a Angular. Actualmente estoy implementando un sistema de caché para agilizar la carga de artículos y mejorando el responsive design. El desarrollo es progresivo y utiliza Google Drive como base de datos para almacenar el contenido.
